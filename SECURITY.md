# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Use [GitHub's private vulnerability reporting](https://github.com/ftaeger/rustylight-server/security/advisories/new) instead. This lets us coordinate a fix before public disclosure.

Expect a response within 7 days. If the report is confirmed, we will work on a fix and coordinate a disclosure timeline with you.
